# Описание проекта

## Учебный проект по работе с реляционными базами данных и языком SQL.

## В рамках проекта выполнены задания по четырём предметным областям:

1. Транспортные средства
2. Автомобильные гонки
3. Бронирование отелей
4. Структура организации

## Для каждой базы данных представлены скрипты создания таблиц, наполнения тестовыми данными и отдельные SQL-запросы для решения поставленных задач.

## Выполненные задачи

1. Транспортные средства
Поиск спортивных мотоциклов по заданным критериям.
Анализ автомобилей, мотоциклов и велосипедов с объединением результатов.

2. Автомобильные гонки
Анализ средней позиции автомобилей.
Поиск лучших автомобилей в каждом классе.
Определение лучшего автомобиля среди всех классов.
Анализ классов автомобилей по результатам гонок.
Сравнение автомобилей относительно средних показателей класса.

3. Бронирование отелей
Анализ клиентов с несколькими бронированиями.
Подсчёт расходов клиентов на бронирование.
Определение предпочтений клиентов по типам отелей.

4. Структура организации
Рекурсивный поиск сотрудников и их подчинённых.
Подсчёт количества задач и подчинённых сотрудников.
Анализ менеджеров и организационной структуры компании.

## Запуск проекта
1. Создать базу данных в MySQL.
2. Выполнить файл scheme.sql из нужной директории в папке databases.
3. Выполнить файл data.sql для заполнения таблиц тестовыми данными.
4. Запустить SQL-запросы из папки solutions.
5. Сравнить результаты выполнения запросов с ожидаемыми результатами задания.
